# dev-registry-domain

### How to run

``` 
mvn compile exec:java
```

### Run unit tests

``` 
mvn clean test -P dev
```
### Integration tests
With ruby, rspec and Airborne (https://github.com/brooklynDev/airborne) installed.

``` 
rspec integration_tests.rb
```
